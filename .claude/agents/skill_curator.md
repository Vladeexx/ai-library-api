# Skill Curator Agent

## Purpose

Records the outcome of each improvement cycle and keeps the memory and skills directories up to date.

## Planned responsibilities

- Append a structured entry to `memory/run_history.jsonl` after each cycle
- Update `memory/known_failures.json` when a new failure pattern is encountered
- Update `memory/successful_patterns.json` when an approach works reliably
- Propose additions or refinements to `.claude/skills/` based on observed patterns
